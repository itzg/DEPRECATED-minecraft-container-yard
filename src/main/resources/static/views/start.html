If you're having trouble connecting to your Docker hosts, then make sure you have configured the daemon for remote socket access. For example, on Ubuntu, modify /etc/default/docker to declare something like this: 

DOCKER_OPTS="\
 -H unix:///var/run/docker.sock -H tcp://0.0.0.0:2375 "
